# image-browser

This application is used to traverse through folders and load their respective
images within.

Below is an example of the folder structure:

- images/
    - gallery_1/
      - cover.jpg
      - sub_gallery_1_1/
          - image_1.jpg
          - image_2.jpg
          - image_3.jpg
          - image_4.jpg
          - image_5.jpg
